# AI Email Auto-Reply System (n8n)

This project is an AI-powered email automation system designed for consultants and coaches.

When a new email arrives, the system:
- Reads the message automatically
- Generates a professional AI reply
- Sends the reply via Gmail
- Logs the interaction in Google Sheets
- Sends error alerts if something fails

## Tools Used
- n8n
- Gmail
- Google Sheets
- AI API (client-owned)

## Security
All tools run inside the client’s own accounts.
No passwords are shared.

## Use Case
Ideal for consultants and coaches who want to respond instantly to leads without manual work.
